# Tripleten web_project_api_full

# Descrição

Projeto monolitico com criação de rotas, rotas protegidas e redirecionamentos, conexão com base de dados e armazenamento local, manipulação de token e autorização e autenticação

# Stack (MERN)

Utilização de React e React-Router, projeto iniciado com método CRA, JavaScript, Autentição e Autorização e criação de API, incluindo módulos como Node, Express, MongoDB, Mongoose, Bcrypt, Dotenv, JSONWebToken, Validator, Celebrate, Joi, Winston, Express-Winston

# Funcionalidades (CRUD)

- Signup/Signin
- Rotas protegidas
- Edição de perfil (nome, descrição e avatar)
- Adição de cartões
- Like/unlike e remoção de cartões
- Server na nuvem com conexão a base de dados
